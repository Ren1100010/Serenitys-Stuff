Gym Cat

Context

I came up with this idea as I was brainstorming personal projects I could do during my summer at AFH. I really like cats and going to the gym so I put them together!

Objective

The player, a human-like cat, has to go to the gym and do exercises to gain strength points to level up!

Vision

I want this game to be able to be played on different platforms, especially ON mobile! Gym Cat has potential to touch the hearts of many different audiences!

How to Play

There are green buttons in each location to navigate the map. Each minigame has its own set of controls, from clicking the mouse to using the keyboard!
