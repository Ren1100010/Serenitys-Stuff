Willy Cat: Microgames

What Are Microgames?

Willy Cat is a collaboration project between the Animation and Coding sub-studios where we all make a series of microgames. Microgames are easy-to-play games that don’t have extreme complexity in terms of flow and gameplay.

Context

This was the first project I ever coded when I started in Creative Technology Studio at AFH! The idea only hit me when I started thinking about concepts I learned in my Java programming class. Then the rest of the game developed from there!

Objective

The player has to form the letter A using the grid to fill in the word “CAT”. Did I mention I like cats?

How to Play

Using a game controller, the user navigates and toggles the grid pieces “on” until the selected pieces form the letter “A”.

Vision

I initially had a meme-inspired vision for this project but I had to change it. This vision was beyond my coding capabilities using the Phaser library. Hopefully I can implement this kind of gameplay in the future!
